# Laptop-Price-Analysis
Laptop Price Analysis Project

This project focuses on analyzing laptop price data to understand the factors that influence laptop pricing in the market. The dataset includes various laptop specifications such as brand, processor, RAM, storage, operating system, screen size, and price.

The main objective of this project is to clean and analyze the data, perform exploratory data analysis (EDA), visualize key patterns, and identify important features that affect laptop prices. Basic machine learning techniques are also used to predict laptop prices based on specifications.

# Tools & Technologies Used

Python

Pandas, NumPy

Matplotlib, Seaborn

Scikit-learn

Jupyter Notebook

# Key Tasks Performed

Data cleaning and preprocessing

Exploratory Data Analysis (EDA)

Visualization of price trends and feature relationships

Feature selection and encoding

Laptop price prediction using machine learning models

Model evaluation and performance comparison

# Project Objective

To analyze laptop price trends and build a predictive model that estimates laptop prices based on their specifications.
